# 🌌 Oráculo dos Livros

Uma experiência literária interativa desenvolvida com **React + Vite**, criada para transformar a escolha da próxima leitura em algo mágico, cinematográfico e divertido.

O projeto funciona como um verdadeiro **oráculo literário**: o usuário escolhe um gênero, ativa o sorteio e acompanha uma roleta animada de capas até que um livro seja revelado como sua próxima leitura.
Confira no site: https://oraculodoslivros.netlify.app/

Toda a aplicação foi desenvolvida com foco em:
- experiência visual;
- animações fluidas;
- estética moderna;
- imersão;
- responsividade;
- identidade visual única.

O resultado é uma plataforma inspirada em:
- experiências premium;
- estética booktok/bookcore;
- interfaces neon futuristas;
- aplicativos interativos modernos.

---

# ✨ Funcionalidades

## 📚 Sorteador inteligente de livros

O usuário pode escolher entre dezenas de gêneros literários como:

- Terror
- Romance
- Fantasia
- Distopia
- Cyberpunk
- Suspense Psicológico
- Mitologia
- Filosofia
- Mangá
- Cozy Books
- Young Adult
- Horror Cósmico
- e muitos outros.

Após selecionar um gênero, o Oráculo dos Livros realiza um sorteio dinâmico utilizando dados reais da Open Library API.

---

## 🎰 Roleta cinematográfica

O sorteio foi pensado para criar uma experiência imersiva.

As capas:
- aparecem rapidamente;
- deslizam pela tela;
- desaceleram suavemente;
- até parar lentamente no livro final escolhido.

O comportamento é inspirado em:
- roletas;
- cassinos;
- Spotify Shuffle;
- Netflix carousel;
- sistemas de loot/gacha.

---

## 🌙 Tema Dark Neon + ☀️ Light Mode

O projeto possui dois modos de uso:

### 🌙 Dark Neon
Visual futurista com:
- glow neon;
- tons de rosa e verde;
- fundo cinematográfico;
- atmosfera mística.

### ☀️ Light Mode
Versão clara e elegante com:
- tons suaves;
- leitura confortável;
- visual clean moderno.

A preferência do usuário é salva automaticamente no navegador.

## 📖 Dados dinâmicos

Os livros são carregados em tempo real utilizando a:

# Open Library API

Permitindo:
- milhares de livros;
- capas automáticas;
- autores;
- descrições;
- links externos.

Tudo isso sem necessidade de cadastrar manualmente cada obra.

# 📦 Instalação

Clone o projeto:

```bash
git clone https://github.com/SEU-USUARIO/oraculo-dos-livros.git
```

Entre na pasta:

```bash
cd oraculo-dos-livros
```

Instale as dependências:

```bash
npm install
```

Inicie o projeto:

```bash
npm run dev
```
